- Rodar no celular
- Instale o app Expo Go (Android/iOS).
- Escaneie o QR Code.
- O app abre seu projeto diretamente no celular.
- Rodar em emulador
- Para Android: instale Android Studio e configure um emulador.
- Para iOS: precisa de Mac com Xcode.
- No terminal, você pode rodar:

Bash
npm run android

Bash 
npm run ios
